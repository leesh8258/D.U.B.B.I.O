- 개발중인 프로젝트로 지속적인 수정이 있을 예정입니다

## 1. 게임소개
- 소개
  - 플레이어는 D.U.B.B.I.O 가 되어 테러범을 방해, 추적, 최종적으로 테러에 대한 디테일을 파악해내 테러범을 막아내야합니다.
  - 미니게임을 통해 단서를 얻고 단서를 사용하여 범인을 추적하여 최종적으로 테러가 발생할 "장소" 와 "무기"를 찾아내는 것이 목표인 게임입니다.

- 개발 팀: RIMU GAMES
  
- 장르: 포인트앤클릭, 퍼즐
  
- 개발 일자: 2025.08 ~ (진행중)
  
- 개발 환경: C#, Unity
